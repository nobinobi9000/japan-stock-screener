プライバシーポリシー

日本株スクリーニングサービス（以下「本サービス」）は、 利用者の個人情報の取扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本サービスでは、以下の情報を収集します：

情報の種類 収集目的
Discord アカウント情報 Discord 通知サービスの提供
決済情報（Stripe経由） 有料プランの決済処理
アクセスログ サービス改善・セキュリティ確保

2. 情報の利用目的

収集した個人情報は、以下の目的で利用します：

  • スクリーニング結果の通知
  • プラン管理・請求処理
  • サービスの改善・新機能の開発
  • お問い合わせへの対応
  • 規約違反行為への対応

3. 情報の第三者提供

当方は、以下の場合を除き、個人情報を第三者に提供しません：

  • 利用者の同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合

3.1 外部サービスの利用

本サービスは、以下の外部サービスを利用しています：

  • Discord: 通知配信
  • Stripe: 決済処理
  • GitHub Pages: Webホスティング

これらのサービスには、それぞれのプライバシーポリシーが適用されます。

4. 情報の保管・管理

当方は、個人情報への不正アクセス、紛失、破損、改ざん、漏洩等を防止するため、 適切な安全管理措置を講じます。

5. Cookie等の使用

本サービスでは、サービス改善のため、以下の技術を使用する場合があります：

  • Cookie
  • Google Analytics（予定）

6. 個人情報の開示・訂正・削除

利用者は、当方に対して以下を請求できます：

  • 保有する個人情報の開示
  • 個人情報の訂正・追加・削除
  • 個人情報の利用停止・消去

請求は、Discord サーバー内のサポートチャンネル、 または GitHub Issues よりお願いします。

7. プライバシーポリシーの変更

当方は、必要に応じて本ポリシーを変更することがあります。 変更後のポリシーは、本サービス上に掲載された時点で効力を生じます。

8. お問い合わせ

個人情報の取扱いに関するお問い合わせは、 Discord サーバー内のサポートチャンネル、 または GitHub Issues よりお願いします。
