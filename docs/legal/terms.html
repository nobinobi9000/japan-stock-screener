利用規約

重要: 本サービスを利用する前に、必ずこの利用規約をお読みください。 本サービスを利用することにより、本規約に同意したものとみなされます。

第1条（適用）

本規約は、日本株スクリーニングサービス（以下「本サービス」）の利用に関する条件を、 本サービスを提供する運営者（以下「当方」）と利用者との間で定めるものです。

第2条（サービス内容）

本サービスは、以下の機能を提供します：

  • 日本株式市場のテクニカル分析に基づくスクリーニング結果の提供
  • Discord による通知サービス
  • HTMLレポートの閲覧
  • 過去のスクリーニング結果の閲覧（プランによる）

第3条（利用登録）

3.1 登録

本サービスの利用を希望する者は、Discord サーバーへの参加により利用登録を行うものとします。

3.2 登録情報の管理

利用者は、登録情報（Discord アカウント）を正確かつ最新の状態に保つ責任を負います。

第4条（料金及び支払い）

4.1 プラン

本サービスには以下のプランがあります：

  • 無料版: 厳選3銘柄の通知
  • ベーシック: 全銘柄HTMLレポート（¥980/月）
  • プレミアム: 30日分アーカイブ + チャート（¥2,980/月）

4.2 支払方法

有料プランの支払いは、Stripe を通じたクレジットカード決済により行います。

4.3 返金

一度支払われた料金は、サービスの性質上、原則として返金いたしません。 ただし、当方の責めに帰すべき事由による場合はこの限りではありません。

第5条（免責事項）

5.1 投資判断

本サービスは投資助言サービスではありません。 提供される情報は過去のテクニカル指標に基づく参考情報であり、 将来の投資成果を保証するものではありません。

5.2 投資リスク

株式投資にはリスクが伴います。投資判断は利用者ご自身の責任で行ってください。 本サービスの利用により生じた損失について、当方は一切の責任を負いません。

5.3 情報の正確性

当方は、本サービスで提供される情報の正確性、完全性、有用性について保証しません。 データ取得元（yfinance等）の不具合により、情報が不正確となる場合があります。

5.4 サービスの中断・停止

当方は、以下の場合にサービスを中断・停止することがあります：

  • システムメンテナンス
  • データ提供元の障害
  • 天災・事変その他の不可抗力
  • その他、運営上必要と判断した場合

第6条（禁止事項）

利用者は、以下の行為を行ってはなりません：

  • 虚偽の情報を登録すること
  • 第三者になりすますこと
  • 本サービスの運営を妨害すること
  • スクリーニング結果を商用目的で第三者に提供すること
  • リバースエンジニアリング、解析、改変等を行うこと
  • 法令または公序良俗に違反する行為

第7条（知的財産権）

本サービスに関する著作権、商標権その他の知的財産権は、 当方または正当な権利者に帰属します。

第8条（個人情報の取扱い）

当方は、利用者の個人情報を プライバシーポリシー に従って適切に取り扱います。

第9条（利用停止・解除）

9.1 当方による停止

当方は、利用者が本規約に違反した場合、 事前の通知なくサービスの利用を停止または登録を抹消できます。

9.2 利用者による解約

利用者は、Discord サーバーからの退出により、 いつでも本サービスの利用を終了できます。

第10条（規約の変更）

当方は、必要に応じて本規約を変更できるものとします。 変更後の規約は、本サービス上に掲載された時点で効力を生じます。

第11条（準拠法・管轄裁判所）

本規約の解釈は日本法に準拠し、本サービスに関する紛争については、 東京地方裁判所を第一審の専属的合意管轄裁判所とします。

第12条（お問い合わせ）

本規約に関するお問い合わせは、Discord サーバー内のサポートチャンネル、 または GitHub Issues よりお願いいたします。
